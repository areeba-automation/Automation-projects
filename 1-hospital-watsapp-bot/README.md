
# AI-Powered WhatsApp Hospital Assistant

## Description
This project is a smart WhatsApp chatbot for hospitals that helps patients find doctors and book appointments automatically.

It supports:
- English
- Urdu
- Roman Urdu
- Voice messages

## Features
- AI-based doctor suggestion
- Voice to text (Vosk)
- Multilingual chatbot
- Appointment booking system
- Phone number validation
- Google Sheets integration

## Tech Stack
- Python (Flask)
- Sentence Transformers + FAISS
- Vosk (Speech Recognition)
- WhatsApp Cloud API
- Google Sheets API

## How to Run
1. Install requirements:
   pip install -r requirements.txt

2. Run:
   python app.py

⚠️ Note: Add your own WhatsApp API token and Google credentials before running the project.
